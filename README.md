## How to run the cypress script



### Step 1: Run the app

```
// Assuming you are in root directory
npm i
npm start

```

### Step 2: Update the config for cypress accordingly

* Change Username/Password in `cypress/e2e/loginSSO.cy.js`

### Step 3: Start cypress

```
npm run cypress:on
```
